# cheat — поиск по личной шпаргалке в Markdown

`cheat` — маленький CLI-помощник: ищет по твоему файлу `cheats.md` и показывает только релевантные блоки (команды, шорткаты, заметки).
Идея простая: пока учишься, дополняешь один Markdown-файл, а потом находишь нужное за 2 секунды прямо из терминала.

## Что умеет
- Поиск по ключевым словам (можно несколько слов)
- Показ результатов в отдельном окне-pager (удобно читать длинные ответы)
- Подсветка слов запроса в результатах
- Быстро открыть файл шпаргалки для редактирования

## Где лежит файл шпаргалки
По умолчанию:
- `~/cheats/cheats.md`

Можно переопределить переменной окружения:
- `CHEAT_FILE=/путь/к/файлу.md`

## Установка (macOS / Linux)

1) Положи скрипт `cheat` в:
- `~/.local/bin/cheat`

2) Сделай исполняемым:
```bash
chmod +x ~/.local/bin/cheat
