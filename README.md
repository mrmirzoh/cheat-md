# cheat — поиск по личной шпаргалке в Markdown

`cheat` — маленький CLI-помощник: ищет по твоему файлу `cheats.md` и показывает только релевантные блоки (команды, шорткаты, заметки).
Идея простая: пока учишься, дополняешь один Markdown-файл, а потом находишь нужное за 2 секунды прямо из терминала.

## Что умеет
- Поиск по ключевым словам (можно несколько слов)
- Показ результатов в отдельном окне-pager (удобно читать длинные ответы)
- Подсветка слов запроса в результатах
- Быстро открыть файл шпаргалки для редактирования

## Где лежит файл шпаргалки
По умолчанию:
- `~/cheats/cheats.md`

Можно переопределить переменной окружения:
- `CHEAT_FILE=/путь/к/файлу.md`


## Установка (macOS / Linux)

Создай папку для своих команд и скопируй конфигурационные файлы из этого репозитория:
```bash
mkdir -p ~/.local/bin ~/cheats
curl -fsSL https://raw.githubusercontent.com/mrmirzoh/cheat-md/main/cheat -o ~/.local/bin/cheat
chmod +x ~/.local/bin/cheat
```


Убедись, что `~/.local/bin` есть в PATH

Проверь:
```bash
echo $PATH | tr ':' '\n' | grep -x "$HOME/.local/bin" && echo "OK: PATH настроен"
```

Если ничего не вывело, добавь в ~/.zshrc (для zsh) или ~/.bashrc (для bash):
```bash
export PATH="$HOME/.local/bin:$PATH"
```

Применить изменения:
```bash
source ~/.zshrc`
# или
`source ~/.bashrc
```

Проверь, что команда доступна
```bash
command -v cheat
```
**
```bash
cheat --help
```
Если command `-v cheat` ничего не показывает:
	ты не добавил `~/.local/bin в PATH`
	или не сделал `source ~/.zshrc / source ~/.bashrc`

# Быстрый старт

Создать/открыть файл шпаргалки
```bash
cheat --edit
```

Добавить в `cheats.md` пару блоков

Скрипт ищет по блокам, которые начинаются с заголовка ## или ###.

Пример:
`### mv
Ключевые слова: переместить переименовать move rename
Команда:`
- `mv old new`

Искать
```bash
cheat mv
cheat ssh ключ
cheat nvim сохранить
```
Как пользоваться окном результата pager

Результаты открываются в “пейджере” (обычно это less).
выйти: `q`
поиск внутри результата: /слово и Enter
следующее совпадение: `n`
предыдущее совпадение: `N`

Команды
`cheat --path` — показать путь к файлу шпаргалки
`cheat --edit` — открыть файл шпаргалки в редакторе
`cheat <слова...>` — поиск по шпаргалке

Советы по формату cheats.md
`Делай один блок на одну тему/команду
Внутри блока добавляй “ключевые слова” с синонимами, чтобы поиск находил быстрее
Чем меньше воды, тем лучше (ты же это для себя пишешь)
`
Удалить
```bash
rm -f ~/.local/bin/cheat
```
